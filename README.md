# 🍕 QuickBite - Food Delivery App

A lightweight food delivery application for neighbourhood food ordering with real-time cart management and geolocation features.

## Features

- **Browse Menu**: View available food items across multiple categories (Pizza, Burgers, Sushi)
- **Category Filtering**: Filter food items by category or view all
- **Shopping Cart**: Add items to cart with automatic quantity tracking
- **Real-time Total**: Live calculation of order total
- **Geolocation**: Automatic detection of user location for delivery
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Geolocation API

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Allow location access when prompted (optional)
4. Browse food items and add to cart
5. Click "Checkout" to place your order

## Usage

### Browsing Food
- Click category buttons to filter items
- Click "All" to view the complete menu

### Adding to Cart
- Click "Add to Cart" on any food item
- Cart updates automatically with quantity and total
- Multiple clicks increase quantity

### Checkout
- Review items in the cart sidebar
- Click "Checkout" to complete the order
- Cart clears after successful checkout

### Location
- App requests location permission on load
- Coordinates display in the header
- Used for delivery address tracking
